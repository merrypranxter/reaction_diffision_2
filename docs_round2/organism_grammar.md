# Organism Grammar

This repo needs a grammar for composing fake organisms from morphogenesis fields.

## Base sentence

```txt
seed → field → instability → growth → symmetry → boundary → ornament → material
```

## Example

```txt
random seed noise
→ multi-scale Turing field
→ local activator/inhibitor competition
→ radial fungal growth bias
→ 9-fold cyclic symmetry
→ soft circular boundary
→ diatom pore/rib ornament
→ opalescent silica material
```
