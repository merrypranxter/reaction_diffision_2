# Morphogenesis Overview

Morphogenesis means form-making. In biology, it refers to the processes by which organisms develop shape: spots, stripes, branching, segmentation, folds, vascular networks, shells, bones, leaves, fungal colonies, and microscopic ornament.

For generative art, morphogenesis is a family of engines that creates images by letting local rules become visible at larger scales.

## Core visual ingredients

- local activation
- long-range inhibition
- diffusion
- growth bias
- branching
- segmentation
- repeated symmetry
- scale competition
- curvature
- boundary pressure
- nutrient gradients
- chemical fronts
- field memory

## Why go beyond Gray-Scott?

Gray-Scott is beautiful but often collapses into recognizable blob language. This repo expands the vocabulary:

| Engine | Visual tendency |
|---|---|
| Gray-Scott | spots, worms, coral blobs |
| Turing patterns | animal coats, lace, stripes |
| Multi-scale Turing | cellular ornament, nested biological detail |
| Cyclic symmetry | mandalas, shells, diatoms, ornamental organisms |
| Growth fields | directional veins, folds, stretched tissue |
| Slime mold coupling | path networks, trails, adaptive tendrils |
| Fungal colony fields | radial bloom, nutrient rings, mycelial fuzz |
| Structural color masks | iridescent bands, beetle shells, oil-slick membranes |
| Crystalline overlays | mineral ribs, facets, geodesic scar tissue |

## Visual metaphor

The image should feel like it is being grown, not drawn.

The viewer should feel like they are looking at:

- a petri dish that discovered architecture
- a fossil that grew a user interface
- a stained-glass organism
- an embryonic cathedral
- a map made by mold, shell, and lightning arguing in a wet basement
