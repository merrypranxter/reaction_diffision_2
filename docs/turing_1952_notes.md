# Turing 1952 Notes

Alan Turing's morphogenesis work introduced a root idea for computational pattern formation: stable-looking biological patterns can emerge from interacting substances diffusing at different rates.

## Generative-art translation

You do not need to simulate real chemistry to steal the visual engine.

Important ideas:

- local reaction
- diffusion over space
- instability from competing rates
- small random differences becoming visible structure
- spots, stripes, waves, rings, patches
- symmetry breaking
- emergence from repeated local updates

## Art controls

| Biological idea | Art control |
|---|---|
| morphogen concentration | scalar field brightness |
| diffusion rate | blur radius or smoothing speed |
| activator | local brightening / amplification |
| inhibitor | broader suppression |
| instability | contrast growth |
| boundary condition | edge behavior / tiling / mirror |
| developmental time | iteration count |
| tissue geometry | coordinate warp or mask |
