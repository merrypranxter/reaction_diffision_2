# Repo Interoperability

This repo is meant to be plugged into the user's existing visual-code ecosystem.

## Input from other repos

### `reaction_diffusion`
Baseline chemical field logic.

### `fungi`
Growth metaphors, colony presets, mycelial branching ideas.

### `slime_molds`
Adaptive pathfinding, chemoattractant maps, trail reinforcement.

### `diatomic`
Diatom/radiolarian body plans, silica ornament, radial symmetry.

### `structural_color`
Iridescent palette logic based on angle, interference, and thin-film vibes.

### `cellular_automata`
Discrete territory formation, cell states, rule-based transitions.

## Output to style repos

### `art_nouveau_style`

Use:
- contours

- vascular lace
- curling boundaries
- plantlike symmetry

### `mesoamerican_style`

Use:
- quantized regions

### `crystalline`

Use:

- ridge extraction
- hard thresholded facets
- angular overlays
- opal/quartz palettes
