# Structural Color Bridge

Use morphogenesis fields as thin-film or interference color maps.

## Inputs
- scalar field as height
- gradient direction
- curvature
- view angle or fake angle

## Transform
- map height to hue bands
- map gradient to shimmer direction
- map curvature to spectral shift
- mask high-activity regions with iridescence

## Visual words
beetle-shell morphogens, oil-slick tissue, opal membrane, morpho-blue veins, soap-film organism skin
