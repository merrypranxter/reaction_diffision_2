# Fungi Bridge

Use morphogenesis fields as fungal colonies and mycelial textures.

## Inputs
- radial growth mask
- multi-scale field
- trail/ridge extraction
- nutrient spots

## Transform
- make colony fronts visible
- add ring pulses
- convert ridges into mycelial threads
- use fuzzy boundary decay
- layer spores over high-activation zones

## Visual words
mycelial lace, spore halo, fairy ring interference, lichen crust field, cordyceps geometry
