# Crystalline Bridge

Use morphogenesis fields as mineralized biological heightmaps.

## Inputs
- heightmap
- gradient map
- high-value ridge mask
- Voronoi/cell overlay

## Transform
- convert smooth tissue into facets
- trace ridges as quartz veins
- apply opal/structural color palette
- add hard specular bands
- combine soft biology with mineral armor

## Visual words
opal embryo armor, quartz fungus, geode tissue, calcite membrane, faceted morphogenesis, mineralized slime lace
