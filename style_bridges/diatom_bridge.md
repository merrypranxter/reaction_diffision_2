# Diatom Bridge

Use morphogenesis fields as silica ornament systems.

## Inputs
- cyclic symmetric field
- radial rings
- pore thresholds
- ridge/contour extraction

## Transform
- enforce radial or bilateral symmetry
- create pore fields from thresholds
- preserve ribs as raised silica
- apply glass/opalescent palette
- render like microscopic jewelry

## Visual words
silica frustule, opal plankton wheel, glass pore halo, radiolarian shield, microscope cathedral
