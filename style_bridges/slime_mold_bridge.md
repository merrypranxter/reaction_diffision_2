# Slime Mold Bridge

Use morphogenesis fields as trail maps for adaptive slime growth.

## Inputs
- scalar field
- gradient map
- agent trail density
- chemoattractant spots

## Transform
- agents follow morphogen ridges
- trails reinforce high-value corridors
- low-value zones evaporate
- extract trail network as nerves/veins

## Visual words
adaptive slime graph, chemoattractant lace, nutrient trail cathedral, golden vein crawl, living path map
