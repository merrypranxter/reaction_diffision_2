# Shader Prompt Modules

## WebGPU compute shader request

Create a WebGPU browser demo implementing a multi-scale Turing morphogenesis engine. Use ping-pong floating point textures. For each pixel, evaluate multiple activator and inhibitor radii, select the best response scale, update the scalar field, normalize or contrast-shape the result, then render with a cyclic symmetry fold and an iridescent biological palette. Include controls for scale count, inhibitor multiplier, step size, symmetry order, domain warp, contrast, and export PNG/JSON buttons.

## p5.js fallback request

Create a single-file p5.js CPU demo that runs in the browser without build tooling. It should simulate multi-scale Turing patterns on a small grid, render upscaled to canvas, include reset/save controls, and use a vivid fungal-opal palette. Keep the code readable and heavily commented.

## Style bridge request

Take a generated morphogenesis mask and transform it into a style-specific output for `art_nouveau_style`, `mesoamerican_style`, or `crystalline`. Use the mask as the underlying biological field, not as generic texture. Explain how thresholds, contours, ridges, and gradient directions map into the final style.
