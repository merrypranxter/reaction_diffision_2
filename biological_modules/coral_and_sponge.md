# Coral and Sponge Module

## Visual target

Porous tissue, branching coral, holes, ridges, living limestone.

## Engine stack

```txt
reaction field
+ growth front
+ pore threshold
+ branching SDF
+ heightmap normals
```

## Pattern tricks

- Use high-value thresholds as raised ridges.
- Use low-value holes as pores.
- Run cyclic symmetry for coral medallions.
- Use growth bias upward or outward.
- Apply crystalline overlay for mineral skeleton.

## Presets

- sponge cathedral
- coral nervous system
- porous moon reef
- radioactive aquarium fossil
- soft limestone organism
