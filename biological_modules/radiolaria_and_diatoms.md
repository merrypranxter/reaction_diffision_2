# Radiolaria + Diatom Module

## Visual target

Microscopic silica organisms that look like the universe got bored and invented jewelry.

## Ingredients

- cyclic symmetry
- pores
- ribs
- concentric rings
- radial spokes
- glass/opalescent material
- high relief heightmap
- lattice perforations

## Algorithm stack

```txt
cyclicFold(uv, symmetry)
+ multiscaleTuring
+ radial ring oscillator
+ pore threshold
+ rib gradient extraction
+ silica material shader
```

## Extra ornaments

- spoke count mismatch
- double symmetry overlay
- prime-number sector count
- broken radial seam
- small pores inside large pores
- mineralized edge halo
